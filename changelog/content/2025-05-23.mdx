The original **Backstage Pro v1** release introduced professional-grade timing control
for live events and broadcast environments. This version represents the foundation
of DYTOR's product direction.

Legacy website:
https://bsp-drab.vercel.app

## Backstage Pro v1 Overview

**Professional Broadcast Stage Panel** - the ultimate live event management solution.
Built for broadcast professionals who demand precision, reliability, and seamless
operation under pressure.

## Core Capabilities

- Dual timer system with real-time display, pause/resume, and configurable alerts.
- Event scheduling with drag-and-drop queue management and auto-fill presets.
- Message broadcasting with preset options, flash effects, and visibility control.
- Multi-display support for primary control + secondary display screens.
- Role-based access for Admin, Queue Manager, Speaker, and Viewer workflows.

## Feature Highlights

**Dual-Display Architecture**  
Dedicated operator control interface with separate presenter display and zero-latency
synchronization across screens.

**Broadcast Messaging System**  
Instant communication pipeline to talent and crew with programmable templates and
silent cuing for uninterrupted shows.

**Production Documentation Suite**  
Comprehensive event logging with automatic timestamping, segment tracking, and
professional PDF reports.

## Professional Experience

**Precision Timing Engine**  
Frame-accurate timing with millisecond precision, count-up modes, and dynamic
segment management.

**Operator Efficiency Suite**  
Complete keyboard command system designed for high-pressure live environments.

**Adaptive Visual Design**  
Interface themes optimized for varied lighting conditions and studio use.

**Accessibility**  
WCAG 2.1 AA compliant interface with full screen reader and keyboard support.
